# Hangman-VHDL-Spartan3
Implementação completa do jogo da Forca em VHDL para o kit de desenvolvimento FPGA Spartan-3A. O projeto integra uma interface de teclado PS/2 para entrada de dados e um display LCD 16x2 para feedback visual do jogo, incluindo lógica de controle de vidas e banco de palavras.
